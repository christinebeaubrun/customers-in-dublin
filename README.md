Locate nearest customers accepts the following args
/* 
  
	  -- startLatt [ Float ]
	  -- endLong [ Float ]
	  -- customers [ Array ]
	  ---- customer [ Object ]
	  ------ longitude [ Float ]
	  ------ latitude [ Float ]
	  ------ name [ String ]

  Returns array

*/

Please have node installed in advance
Sample code to run

cd customers-in-dublin
node js/customers-near-long-lat 53.3381985 -6.2592576 sample-data.js
